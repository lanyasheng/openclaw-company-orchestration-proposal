# Validation 文档区

这里存放的是：
- P0 审计结论
- readiness review
- bridge simulator
- human-gate / callback / task registry 契约
- 各类 repo-local POC 证明材料

## 定位

这些文档**非常重要**，因为它们提供证据；但它们**不再承担仓库主线叙事**。

主线请先看：
1. `../executive-summary.md`
2. `../openclaw-company-orchestration-proposal.md`
3. `../architecture-layering.md`
4. `../validation-status.md`
5. `../roadmap.md`

## 原则

- 主线文档回答“总方案是什么”
- validation 文档回答“某个判断的证据是什么”
- 不允许再让局部 POC 覆盖总方案口径
